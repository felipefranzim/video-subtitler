# Video Subtitler

Objetivo: criar um sistema que, ao realizar o upload de um vídeo com áudio em inglês, utilize inteligência artificial para colocar legendas em inglês na parte inferior do vídeo e legenda em português, de forma simultânea, na parte superior do vídeo.

Pode ser criada uma prova de conceito para testar as abordagens (apenas se for necessário) utilizando qualquer linguagem de programação ou tecnologia.

Se for necessário, podemos usar uma api key da OpenAI.

Agentes não devem sair escrevendo códigos e editando arquivos sem autorização. Ao invés disso, deve sempre buscar mostrar o caminho e os códigos a serem desenvolvidos, com o máximo de detalhes possíveis e não sendo razo nas explicações.
